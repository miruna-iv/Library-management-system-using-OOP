# Library-management-system-using-OOP
This project is an advanced library management solution developed in C++. It is designed to demonstrate core and advanced **Object-Oriented Programming (OOP)** concepts while providing a functional tool for managing book inventories, authors, and client databases. The system uses SQLite for robust data persistence.


##Key Features
- **Multi-level Hierarchy:** Manages different types of publications (Standard Books, Rare Books, etc.) through a structured inheritance tree.
- **Persistence Layer:** Fully integrated with an SQLite database (`librarie.db`) to ensure data is saved across sessions.
- **CRUD Operations:** Complete Create, Read, Update, and Delete functionality for books, authors, and customers.
- **Transaction Simulation:** Handles client budgets, genre preferences, and book acquisitions.
- **Interactive CLI:** A user-friendly command-line interface for seamless navigation.

##Technical Implementation & OOP Concepts
The project strictly adheres to the following principles:
- **Encapsulation:** Data members are protected and accessed through getters, setters, and constructors.
- **Polymorphism:** Use of **virtual functions** and base-class pointers to enable dynamic dispatch.
